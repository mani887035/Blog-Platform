Blog Platform with Comments

A simple and responsive blogging platform built using HTML, CSS, and JavaScript. Users can create blog posts, view posts, and add comments. All data is stored locally using the browser's Local Storage, making it easy to deploy on GitHub Pages without any backend setup.

Installation
Clone Repository
git clone https://github.com/mani887035/Blog-Platform.git
Open Project
cd Blog-Platform

Open index.html in your browser.

No additional setup or dependencies are required.

Deployment on GitHub Pages

This project is deployed using GitHub Pages, which hosts static websites directly from a GitHub repository.

Steps
Push project files to GitHub.
Open Repository Settings.
Navigate to Pages.
Select:
Source: Deploy from branch
Branch: main
Folder: /root
Save changes.

GitHub Pages will generate a public URL for your website.
